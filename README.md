# Trainly — Fitness & Client Management App

**Trainly** is a modern Android application designed for fitness trainers to manage their clients, track progress, and organize workout schedules. Built with performance and scalability in mind using the latest Android development standards.

## 🚀 Key Features (In Progress)
* **Client Profile Management:** Comprehensive screen to view and edit client personal data, fitness goals, and notes.
* **Dynamic UI:** Smooth and reactive interface built entirely with Jetpack Compose.
* **Offline Support:** Robust local data storage using Room database.
* **Clean Architecture:** Separation of concerns using MVVM pattern.

## 🛠 Tech Stack
* **Language:** Kotlin
* **UI Framework:** Jetpack Compose
* **Local Database:** Room
* **Architecture:** MVVM (Model-View-ViewModel)
* **Asynchrony:** Kotlin Coroutines & Flow
* **Dependency Injection:**  Koin
